mdl_assign
Contém dados sobre as tarefas (assignments) criadas nos cursos.

mdl_block
Armazena os blocos (widgets ou painéis adicionais) que podem ser posicionados nas páginas do Moodle. 

mdl_book
Guarda os livros (book resources) que podem ser criados para fornecer conteúdo estruturado nos cursos.

mdl_capabilities
Define as capacidades (permissões) que podem ser atribuídas a papéis. 

mdl_chat
Regista as sessões de chat, ou seja, os dados relativos à comunicação em tempo real entre utilizadores.

mdl_choice
Contém dados relativos a atividades de escolha (polls ou enquetes), onde os alunos escolhem uma opção.

mdl_cohort
Guarda informações sobre os cohorts, que são grupos de utilizadores agrupados para gestão em massa. 

mdl_comments
Armazena os comentários feitos em diversas partes da plataforma (p.ex. em fóruns, trabalhos, etc.).

mdl_communication
Geralmente regista dados de comunicação interna (eventos, registos de mensagens ou outras interações).

mdl_competency
Contém definições e informações relacionadas com competências, usadas para acompanhar o desenvolvimento e progresso dos alunos. 

mdl_config
Guarda as configurações gerais da instância do Moodle. 

mdl_context
Define os contextos (como cursos, módulos, etc.) e serve de base para a atribuição de papéis e permissões. 

mdl_course
Regista os cursos disponíveis na plataforma, com todas as suas configurações e metadados. 

mdl_data
Pertence à atividade “data” (bases de dados criadas pelos utilizadores) e guarda a definição dessas bases. 

mdl_enrol
Define os métodos de inscrição nos cursos (manual, auto-inscrição, etc.). 

mdl_event
Armazena os eventos (como os do calendário) e outras marcações temporais dos cursos. 

mdl_feedback
Guarda os dados das atividades de inquérito ou feedback, onde os alunos respondem a questionários. 

mdl_files
Regista os ficheiros armazenados na plataforma (upload, downloads, etc.). 

mdl_folder
Relaciona-se com os recursos de pastas, onde se agrupam ficheiros. 

mdl_forum
Contém dados dos fóruns de discussão, incluindo tópicos e interações. 

mdl_glossary
Armazena os glossários criados nos cursos, com as definições e termos. 

mdl_grade_grades
Guarda as notas obtidas pelos alunos. 

mdl_groupings
Define agrupamentos de grupos, permitindo organizar grupos em conjuntos. 

mdl_lesson
Contém dados sobre as lições, que são atividades interativas estruturadas. 

mdl_lti
Relaciona-se com as integrações LTI (Learning Tools Interoperability), permitindo a ligação com ferramentas externas. 

mdl_message
Regista as mensagens trocadas entre os utilizadores, servindo como base para o sistema de comunicação interna. 

mdl_modules
Lista os módulos (tipos de atividades) disponíveis no Moodle. 

mdl_my_pages
Guarda as páginas personalizadas dos utilizadores ("minhas páginas").

mdl_notifications
Armazena as notificações enviadas aos utilizadores. 

mdl_question
Contém as questões usadas nos bancos de perguntas, em quizzes ou outras atividades. I

mdl_quiz
Guarda as definições dos quizzes, incluindo configurações, tempos, etc. 

mdl_registration_hubs
Relaciona-se com a gestão de registos externos ou hubs de registo. 

mdl_role
Define os papéis (roles) existentes na plataforma, como administrador, professor, aluno, etc.

mdl_scale
Guarda as escalas de avaliação (ex: escalas de notas) que são utilizadas na correção. 

mdl_scorm
Armazena informações sobre os pacotes SCORM, que são conteúdos interativos. 

mdl_sessions
Regista as sessões dos utilizadores (dados de login e atividade).

mdl_stats_daily
Contém estatísticas diárias do Moodle, como logins e atividade global. 

mdl_survey
Guarda as definições das pesquisas (surveys) realizadas na plataforma.

mdl_tag
Armazena os tags (etiquetas) atribuídos a diversos elementos, permitindo categorizar e organizar conteúdos. 

mdl_task_scheduled
Regista as tarefas agendadas (cron jobs) que o Moodle executa periodicamente. 

mdl_user
Contém os dados dos utilizadores, como informações pessoais, emails, etc.

mdl_wiki
Guarda os dados relativos aos wikis criados nos cursos. 

mdl_workshop
Regista as atividades do tipo workshop, onde se realiza avaliação entre pares e revisões.
